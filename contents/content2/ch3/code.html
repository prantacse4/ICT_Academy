কম্পিউটার কোড

কম্পিউটার সিস্টেমি ব্যবহৃত প্রকিটি বর্ণ,অঙ্ক,প্রতীক বা বিশেষ চিহ্নকে আলাদাভাবে CPU কে বুঝানোর জন্য বাইনারি বিটের বিভিন্ন বিন্যাসের অদ্ধিতীয় সংকেত তৈরী করা হয়৷এ অদ্ধিতীয় সংকেতকে কোড বলে৷
প্রয়োগের ক্ষেত্রের উপর ভিত্তি করে বিভিন্ন প্রকার কোড উদ্ভব হয় ৷ যথা:
বিসিডি কোড
BCD এর পূর্ণরুপ হলো Binary Coded Decimal । BCD কোড দ্বারা কেবলমাত্র দশমিক পদ্ধতির প্রতীকগুলো কোড করা হয়৷বর্ণমালার বর্ণ বা অন্য কিছুর জন্য BCD কোড নেই৷ BCD তে ৪ বিট ব্যবহার করে দশমিক সংখ্যার প্রতিটি অঙ্ককে উপস্থাপন করা হয়৷৪ বিটের কারণে BCD কোডের মাধ্যমে ১৬টি ভিন্ন অবস্থা নির্দেশ করে।
ইবিসিডিআইসি কোড
EBCDIC কোডের পূর্ণরুপ হলো Extended Binary Coded Decimal Information Code। এটি একটি ৮ বিটের কোড৷এ কোড দ্বারা ২৫৬ টি অঙ্ক ,অক্ষর বা চিহ্ন প্রকাশ করা যায়।
আলফানিউমেরিক কোড
কম্পিউটারে সংখ্যাসূচক চিহ্নসহ অন্যান্য কিছু বর্ণ বা চিহ্ন ব্যবহার করা হয়৷এসব সংখ্যা ,চিহ্ন বা বর্ণের জন্য যে কোড ব্যবহৃত হয় আলফানিউমেরিক কোড তাকে বলে।একটি সম্পূর্ণ আলফানিউমেরিক কোড এ ২৬ টি ছোট অক্ষর(a-z),২৬টি বড় অক্ষর(A-Z),১০ টি অঙ্ক,৭ টি বিরাম চিহ্ন,কিছু গানিতিক চিহ্নসহ অন্যান্যচিহ্ন রয়েছে।কতগুলো জনপ্রিয় আলফানিউমেরিক কোড হলো অ্যাসকি কোড,ইউনিকোড ইত্যাদি।
অ্যাসকি কোড
এর পূর্ণরুপ হলো American Standard Code For Information Interchange। কম্পিউটার ইনপুট আউটপুট সরন্জ্ঞামের মধ্যে তথ্য স্থানান্তরের জন্য এ কোড ব্যবহার করা হয়।অ্যাসকি কোড ২ ধরণের৷যথাঃ৭-অ্যাসকি ও ৮-অ্যাসকি।অ্যাসকি কোড এ তিন বিটের জোন বিট এবং চার বিটের সংখ্যাসূচক বিট থাকে৷অ্যাসকি-৮ এ অতিরিক্ত একটি Parity বিট থাকে।
ইউনিকোড
এর পূর্ণরুপ হলো Universal Code বা সার্বজনীন কোড৷বিশ্বের সকল ভাষাকে কম্পিউটারে কোডভুক্ত করার জন্য বড় বড় কোম্পানিগুলো একটি মান তৈরী করেছে যাকে ইউনিকোড বলে৷ ইউনিকোড মূলত ২ বাইট বা ১৬ বিটের কোড৷এ কোডের মাধ্যমে ৬৫৫৩৬ টি অদ্বিতীয় চিহ্নকে নিদিষ্ট করা যায়৷ইউনিকোড অন্যান্য স্টান্ডার্ড কোডে পরিবর্তন করা যায়।
